# Life Timer ⏳

A Progressive Web App (PWA) designed to visualize your remaining time based on average life expectancy, acting as a foundational productivity and mortality-awareness dashboard.

## Architecture

This project is built using Vanilla HTML/CSS/JS and is deployed statically via Vercel.

* **`index.html`**: The main view and layout structure.
* **`styles.css`**: All styling, layout, and UI animations.
* **`app.js`**: Core logic for timezone calculations, local storage state, and the ticking interval.
* **`sw.js`**: Service worker handling offline caching.
* **`manifest.json`**: PWA configuration file.
